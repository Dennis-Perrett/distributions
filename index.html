Distribution Cheatsheet

A Cheatsheet for common and uncommon distributions!

Common Distributions

Bernoulli
Bernoulli Distribution
Description

A distribution for a single trial, binary outcome event.

Features:
  • Discrete Data
  • Single Trial
  • Only 2 possible outcomes (Success and Failure)
Example

A single attempt at flipping a coin.

PMF Equation:

$$Pr(X = k | \theta) = \theta^k(1-\theta)^{(1-k)} $$

Binomial
Binomial Distribution
Description

A distribution for a fixed number of binary outcome trials.

Features:
  • Discrete Data
  • Fixed number of trials
  • Each trial has 2 possible outcomes (Success and Failure)
  • Trial outcomes are independent
  • Probability of success is the same with each trial
  • The outcome is the aggregate number of successes
Example

Clinical drug trial. 10 subjects are infected with a virus, and are then given an experimental drug. The outcome can be either recovered (x=1), or not recovered (x=0). If we fix the outcome, we can use ML to estimate the probability. If we fix the probability, we can estimate the number of successes.

PMF Equation:

$$Pr(X = k |n, \theta) = \binom{n}{k}\theta^k(1-\theta)^{(n-k)}$$

Poisson
Poisson Distribution
Description

A distribution for estimating the probability of an event given a rate.

Features:
  • Count of discrete events
  • Non-negative integer numbers
  • Individual events occur at a given rate and independently of each other
  • Fixed amount of time or space in which an event can occur.
Example

Estimating failure rates of artificial heart valves. We collected data over a number of (example: 5) years, and count the number of failures. Let us assume we witnessed failures per year of [7,3,9,8,9]. This gives us a rate of 7.2 per year. From here we can estimate the probability of seeing a certain number of failures (Eg: 2) in the next year (\( Pr(X=2|\lambda) \))

PMF Equation:

$$Pr(X = k |\lambda) = \frac{\lambda^k}{k!}e^{-\lambda}$$

Negative Binomial
Negative Binomial Distribution
Description

A distribution for a single trial, binary outcome event. The can do everything the Poisson can do, and more! Can be useful for modelling the outbreak of diseases, as outbreaks are likely to be influenced by one another. This distribution takes in the same parameters as the poisson, in addition to a "connectedness" parameter.

Features:
  • Count of discrete Data
  • Non-independent events. It can be possible that an event occurs, affecting the probability of a second event occuring.
  • Can model a data-generating process where the variance is greater than the mean.
  • Fixed amount of time (or space) in which the events can occur.
  • Uses \(\kappa\) as a measure of dispersion (small number, less dispersed).
Example

We want to predict the number of flu-infected villagers by modelling the outbreak of a flu in different villages. Suppose each village has some rate \(\theta\) of outbreaks per season. Additionally, villages have some degree of connectedness either via family, commuting etc. This connectedness, or dispersion \(\kappa\) is reflected by \(\theta\), in that villages with higher \(\theta\) are more connected. We calculate an overall rate of infection \(\lambda\) by averaging the individual \(\theta\). If each village has the same level of dispersion, then the Negative Binomial Distribution devolves into the Poisson distribution. By varying \(\theta\) across villages, the variance of the overall distribution increases, and we end up with a wider distribution compared to the Poisson with the variance being given by: $$\lambda + \frac{\lambda^2}{\kappa} \geq var(X_{poisson}) = \lambda,$$ which exceeds the overall variance of individuals with the flu if all towns were identical.

PMF Equation:
Using (inverse) dispersion \(\kappa\):

$$Pr(X = y |\lambda, \kappa) = \frac{\Gamma\left(y + \kappa \right)}{y!\Gamma\left(\kappa + 1 \right)} \left( \frac{\lambda}{\lambda + \kappa} \right)^y \left( \frac{\kappa}{\lambda + \kappa} \right)^\kappa$$

where \(y\) = number of realisations, \(\Gamma\) is the Gamma Function, \(\lambda\) is the overall rate, and \(\kappa\) is the inversion dispersion.

Using \(\theta\)

$$Pr(X = y |r, \theta) = \binom{r+y-1}{r-1}(1-\theta)^y\theta^r$$

Student's-\(t\)
Students-T Distribution
Description

This is the overdispersed cousin of the normal distribution. That is, when the sample variance is greater than what the normal distribution assumes. The Students-t can be considered as a combination of multiple normals with difference variances. Can be used for anything the normal distribution is used for, and more! For example stock returns.

Features:
  • Countinous Data
  • (practically) Unbounded outcomes
  • Outcome is the outcome of a large number of additive factors
  • Data sample composed of heterogeneous individuals
  • Sometimes called the cauchy-distribution
Example

Suppose we want to model the test-score distribution after combining 2 schools with normal distributions and same means but with differing variances. The combination of these schools, while keeping the mean the same, will result in a distribution that is no longer normal. This is mostly evident in the kurtosis or fatter tails.

PDF Equation:

$$Pr(X = x |\mu, \sigma, v) = \frac{\left(\frac{v}{v+\frac{(x-\mu)^2}{\sigma^2}} \right)^{\frac{v+1}{2}}}{\sqrt{v}\sigma B\left(\frac{v}{2}, \frac{1}{2} \right)}$$

where \(v\) is the degrees of freedom.

Normal Distribution
Normal Distribution
Description

The normal distribution we all know and love. Nice bell-shape. Occurs everywhere in nature.

Features:
  • Countinous data
  • Unbounded, or at least practically unbounded.
  • Outcome is the result of large number of additive factors.
Example

Can be used to explain the distribution of body temperatures among people. The distribution requires 2 paramters: \(\mu\), the mean, and \(\sigma\), the standard deviation. We gather these 2 parameters via sampling, and then have a distribution. We can then estimate the probability of a certain person having a specific body temperature.

PDF Equation:

$$Pr(X = x|\sigma, \mu) = \frac{1}{\sqrt{2\pi\sigma^2}}e^{-(\frac{(x-\mu)}{2\sigma})^2}$$

Less-common Distributions

Beta Binomial
Beta-Binomial Distribution
Description

Very similar to the binomial, but makes an additional assumption of different probabilities per sub-trial. In other words, can do everything the binomial does, and more!

Features:
  • Discrete Data
  • Multiple trials (reps), with a fixed number of total trials (sets)
  • Each trial has 2 possible outcomes (Success and Failure)
  • Measures the aggregate number of successes
  • Probability of success varies with each trial
Example

Suppose we want to predict the number of of votes for Party A in a 2 party system across differing states. Each state will likely have their own preferences (probabilities) when voting. We use surveys in each state to estimate each probability \(\theta\). We might assume that for each state, the probability of voting for party A can be modelled by the binomial distribution \(X \sim B(n, \theta)\) where \(n\) is sample size and \(\theta\) is the probability of voting party A for an individual. The overall probability would then be the combination of all state probabilities (weighted average). The beta-binomial distribution can also be motivated via an urn model for positive integer values of \(\alpha\) and \(\beta\), known as the Pólya urn model. Specifically, imagine an urn containing \(\alpha\) red balls and \(\beta\) black balls, where random draws are made. If a red ball is observed, then two red balls are returned to the urn. Likewise, if a black ball is drawn, then two black balls are returned to the urn. If this is repeated \(n\) times, then the probability of observing x red balls follows a beta-binomial distribution with parameters \(n\), \(\alpha\) and \(\beta\).

PMF Equation:

$$Pr(X = y |n, \alpha, \beta) = \binom{n}{y}\frac{B(y+\alpha, n+\beta - y)}{B(\alpha, \beta)}$$

where \(\alpha\) is number of y elements in \(n\), \(\beta\) is number of non-y and \(n\) is the number of trials.

Exponential
Exponential Distribution
Description

Very similar to the binomial, but makes an additional assumption of different probabilities per sub-trial. In other words, can do everything the binomial does, and more!

Features:
  • Continuous data
  • Non-negative data
  • Often used to measure the amount of time or space between certain events that occure independently, and at a constant rate
Example

Modelling the time between new outbreaks of Ebola. We define these outbreaks as not being related to existing outbreaks, and assume that each occurance of an outbreak is the result of non-human-to-human transfer of the virus. We need a rate per time frame, in this case let us use outbreaks per year, which will be our \(\lambda\). Given the lambda, we can now get the probabilities of number of outbreaks per year.

PDF Equation:

$$Pr(X=x|\lambda) = \lambda e^{-\lambda x}$$

Gamma
Gamma Distribution
Description

Similar to the exponential distribution, but for multiple occurances. That is, how long will it take for \(n\) occurances?

Features:
  • Countinous, non-negative data
  • Greater flexibility than the exponential, but more complex
  • Can be used to model the time it takes for \(n\) independent occurances.
Example

Really simply, lets say that we want to estimate how long it will take 3 people to claim their lottery winnings, that is, if 1 person takes 1 day, the second takes 2 days, and the third takes 3 days, the total number of days to claim the prizes is 6. Most people will want to claim their winning asap, but it is unlikely that all 3 will claim instantly. As such, the maximum probability lies a little away from 0. The spread is determined by the rate \(\lambda\) of how many occurances per day (or other time frame) - let's say 0.5 per day. We also need to know the mean time it takes for the winners to claim their prize(s). To do this, we take the sum of the individual means for each winner, which is in our case \(\mu = 1 \times 3 = 3\) which leads us to the function \(\Gamma(\mu = 3, \lambda = 0.5)\).

PMF Equation:
\(\Gamma(\alpha, \beta)\) formulation:

$$Pr(X = x|\alpha, \beta) = \frac{\beta^\alpha}{\Gamma(\alpha)}x^{\alpha-1}e^{-\beta x} $$

\(\Gamma(\kappa, \theta)\) formulation:

$$ Pr(X = x|\kappa, \theta) = \frac{1}{\Gamma(k)\theta^\kappa} x^{k-1} e^{-\frac{x}{\theta}} $$

\(\Gamma(\mu, \sigma)\) formulation:

$$Pr(X = x|\mu, \sigma) = \frac{(\frac{\sigma^2}{\mu})^{-\frac{\mu^2}{\sigma^2}}e^{-\frac{\mu x}{\sigma^2}} x^{\frac{\mu^2}{\sigma^2}-1}}{\Gamma(\frac{\mu^2}{\sigma^2})}$$

Multinomial
Multinominal Distribution
Description

This is the multi-outcome variant of the binomial distribution. We have a fixed number of trials, each with multiple outcomes. For example, instead of modelling the probability of voting for party A in a 2 party system, the Multinomial distribution allows for more than 2 parties.

Features:
  • Discrete Data
  • Multiple trials with a fixed number of total trials
  • Trial outcomes are independent
  • Each trial has more than 2 outcomes
  • Individual outcomes probabilities are not determined by a factor that varies systematically across individuals
  • Probability of obtaining each category is the same in each trial
  • Overall data we record is the aggregate number of outcomes in each category
  • Generalisation of the binomial to handle an arbitrary number of outcome categories
Example

An example could be, modelling the prevalence of blood types of a sample of people. There are multiple blood types, but let us, for the sake of simplicity, assume there are only 3 blood types: A, B, and O. This is relatively straight forward. We would create 3 random variables \(X_A, X_B, X_O\) for each of the blood types, where X=1 if the subject has the given blood type. We know there is an underlying proportion (or probability) for each blood type: \(P_A = \frac{1}{2}, P_B = \frac{1}{3}, and P_O = \frac{1}{6}\). So the question becomes: what is the probability of getting a certain blood type after n trials, given the proportions, or \( Pr(Z_1 = z_1, Z_2 = z_2, ... Z_k = z_k | P_1, P_2,...P_3)\).

PMF Equation:

$$Pr(Z_1 = z_1, Z_2 = z_2, ... Z_k = z_k | P_1, P_2,...P_3) = \frac{n!}{z_1!z_2!...z_3!}P_1^{z_1},P_2^{z_2},...P_k^{z_k}$$

Multivariate Normal
Multivariate Normal Distribution
Description

Similar to the univariate normal distribution but accounts for multiple X random variables

Features:
  • Multivariate distribution
  • Continuous
  • Unconstrained
  • Uses Mean-Vector and Covariance-Matrix
Example

Modelling gene expression of multiple (normally distributed) genes. This model is just the combination of multiple normally-distributed random variables.

PDF Equation:

$$Pr(X=x|\mu\Sigma) = (2\pi)^{-\frac{k}{2}} |\Sigma|^{-\frac{1}{2}}e^{-\frac{1}{2}(x-\mu)'\Sigma^{-1}(x-\mu)}$$ where \(k\) is the number of random variables/dimensions.

Multivariate Students-\(t\)
Multivariate Student's-\(t\) Distribution
Description

Similar to the Univariate-Student's-\(t\) and Multivariate-Normal distributions. The multivariate Student's-\(t\) is the combination of multiple \(t\)-distributed random variables, with more area under the tails than a normal distribution.

Features:
  • Continuous
  • Unbounded
  • Takes in Mean-Vector and Covariance-Matrix
  • More area under tails than normal-distribution
Example

Imagine you are an analyst at a hedge fund, your boss wants you to model the risk of a portfolio containing 2 stocks. The manager says that if all stocks have a simultaneous loss of 5%, then the hedge fund goes bankrupt. The acceptable risk of this is \(\frac{1}{1000}\). You take the daily returns of the 2 stocks and take the multivariate Student's-\(t\) distribution. The area under the curve where both stock returns are sub 5%, is the cumulative probability (risk) of such an event occuring. We settle on the Student's-\(t\) because stock returns are slightly more volatile (more area under the tails) than would be modelled by a normal distribution.

PMF Equation:

$$Pr(X = x|\mu, \sigma, v) = \frac{\Gamma(\frac{v+k}{2})} {\Gamma(\frac{v}{2})v^{\frac{v}{2}}\pi^{\frac{v}{2}}|\Sigma|^{-\frac{1}{2}} \left[1+\frac{1}{v}(x-\mu)'\Sigma^{-1}(x-\mu)\right]^{(v+k)/2}} $$ where \(k\) is the number of random variables/dimension, and \(v\) is the degrees of freedom.

Uniform
Uniform Distribution
Description

A very common choice of uninformative prior for variables constrained between 0 and 1. This distribution has the same probability for all values between \(a\) and \(b\).

Features:
  • Continuous parameters
  • Simple form of beta distribution!
  • Same probability for all values
Example

Best used as an uninformative prior, when we really have no prior information/ideas about the posterior distribution.

PMF Equation:

$$ Pr(\theta|a,b) = \frac{1}{b-a}\ where\ a < b $$

Beta
Beta Distribution
Description

A more flexible version of the uniform distribution. Able to shift and bend this distribution to mimic a wide range of distributions.

The Beta function takes in \(a\) and \(b\). For a non-skewed distribution, have \(a = b\). The magnitude determines the peakedness. To have more mass on the left, have \(a > b\).

Features:
  • Continuous
  • Bounded between 0 and 1
  • \(Beta(1,1)\) is equivalent to the uniform prior
Example

No real example necessary, but distribution allows us to shift the mean, and influence the posterior, between 0 and 1, for example obesity rates. We expect the obesity rate to be below 50%, so we specify a beta prior to have the most density below 50%.

PDF Equation:

$$ Pr(\theta |\alpha, \beta) = \frac{\theta^{\alpha-1}(1-\theta^{\beta-1})}{B(\alpha, \beta)} $$

Logit-Normal
Logit-Normal Prior
Description

A logistic transformation of a normal distribution. This limits the range of the distribution to 0-1. Otherwise this distribution is also highly flexible just like the Beta distribution.

Features:
  • Continuous
  • Bounded between 0-1
  • Encompasses a wide range of priors
Example

No real example necessary, but distribution allows us to shift the mean, and influence the posterior, between 0 and 1, for example obesity rates. We expect the obesity rate to be below 50%, so we specify a logit-normal prior to have the most density below 50%. The idea is to first define a normal distribution with the given properties, then transform it to be bound between 0 and 1.

PMF Equation:

$$ Pr(X=\theta|\mu, \sigma) = \frac{1}{\sqrt{2\pi}(1-\theta)\theta\sigma} \exp{\left( - \frac{(log(\frac{\theta}{1-\theta})-\mu)^2}{2\sigma^2} \right)}$$ where \(\theta\) is \(\frac{1}{1+e^{-a}}\) of \(X \sim logit\text{-}\mathcal{N}(\mu, \sigma)\)

Special Distributions

Dirichlet Distribution
Dirichlet Prior
Description

A multivariate generalisation of the beta distribution. Most relevant, is that the sum of the parameters is equal to 1. The easiest way to understand this distribution is to consider proportions.

Features:
  • Continous Variables
  • Parameters sum to 1
  • Can range from weak to strong prior
Example

Consider we want to estimate what proportion of the population will vote for which political party. In a 2 party system, the distribution may be estimated around 25% for party 1 and 75% for party b. Of course, these are estimates, hence the probability distribution. Now, if we consider a 3 (or more) party system, the proportions may be 25% for party 1, 35% for party 2, and 40% for party 3. For 3 variables, we can imagine this as an equalateral triangle.

PMF Equation:

$$ Pr(\theta | \alpha_1,\alpha_2,...,\alpha_k) = \frac{1}{B(\alpha)}\prod^k_{i=1}\theta_i^{\alpha_i - 1} $$ where $$ B(\alpha) = \frac{\prod^k_{i=1}\Gamma(\alpha_i)}{\Gamma(\sum^k_{i=1 \alpha})}$$

Cauchy Distribution
Cauchy Distribution
Description

A symmetrical distribution, far more flexible that a normal or student's=\(t\)

Features:
  • Continuous
  • Unconstrained
  • Robuster version of the normal or Student's-\(t\), allowing for a greater range of parameter values.
  • A special case of the student's-\(t\) where df = 1
  • Has no mean analytical mean or variance. Just a median
Example

An example can be anything where we would consider a normal, but need more flexibility with respect to extreme values.

PDF Equation:

$$ p(\theta|\theta_0, \gamma) = \frac{1}{\pi \gamma \left( \frac{(\theta - \theta_0)^2}{\gamma^2} + 1\right)} $$

Half-Cauchy Distribution
Half-Cauchy Distribution
Description

A one half cauchy distribution. Very useful for estimating variances, as it only has density for positive values. It is the prefered distribution when compared to Inverse-Gamma and Inverse-\(\chi^2\).

Features:
  • Continuous
  • Constrained between 0 and \(\infty\)
  • No analytical solution for mean or variance.
Example

The crocodile example: If we are trying to estimate the length of fully grown male crocodiles, we are interested in measuring the mean. We can assume that the distribution is normal. Given it is normal, we still need to specify a value for the variance - the problem parameter. We can do this by specifying a prior for the variance. Naturally, the variance cannot be negative, so we need a distribution where negative values have no density. The benefit of the half-cauchy distribution is the gentle slope, which assigns no large density to infeasible parameters.

PDF Equation:

$$ p(\theta|\theta_0, \gamma) = \frac{1}{\pi \gamma \left( \frac{(\theta - \theta_0)^2}{\gamma^2} + 1\right)}\ if\ \theta > 0\ else\ 0$$

Inverse-Gamma Distribution
Inverse-Gamma Distribution
Description

A bounded, and skewed distribution useful for estimating non-negative values. Generated by sampling x from a gamma distribution, and taking \(\frac{1}{x}\)

Features:
  • Bounded
  • Continuous
  • Half-Cauchy is generally preferred
Example

The crocodile example: If we are trying to estimate the length of fully grown male crocodiles, we are interested in measuring the mean. We can assume that the distribution is normal. Given it is normal, we still need to specify a value for the variance - the problem parameter. We can do this by specifying a prior for the variance. Naturally, the variance cannot be negative, so we need a distribution where negative values have no density. The benefit of the inverse-gamma distribution is the skewed nature, meaning we can assign most density around a certain range, with more density toward higher values, and very little near 0.

PMF Equation:

$$ Pr(X=x|\alpha, \beta) = \frac{\beta^\alpha}{\Gamma(\alpha)} x ^ {-\alpha-1} exp\left(-\frac{\beta}{x}\right)$$

Inverse-\(\chi^2\) Distribution
Inverse-\(\chi^2\) Distribution
Description

A bounded, and skewed distribution useful for estimating non-negative values. Generated by sampling x from a \(\chi^2\) distribution, and taking \(\frac{1}{x}\)

Features:
  • Bounded
  • Continuous
  • Half-Cauchy is generally preferred
Example

The crocodile example: If we are trying to estimate the length of fully grown male crocodiles, we are interested in measuring the mean. We can assume that the distribution is normal. Given it is normal, we still need to specify a value for the variance - the problem parameter. We can do this by specifying a prior for the variance. Naturally, the variance cannot be negative, so we need a distribution where negative values have no density. The benefits of the inverse-\(\chi^2\) distribution are the simplicity of parameters and the skewed nature, meaning we can assign most density around a certain range, with more density toward higher values, and very little near 0.

PMF Equation:

$$ Pr(X=x|df) = \frac{2^{-df/2}}{\Gamma(df/2)} x^{-df/2 - 1} e^{-1/2x}$$

Log Normal Distribution
Log Normal Distribution
Description

A very flexible distribution, constrained to positive values. Allows for a large variance.

Features:
  • Continuous
  • Parameters constrained to be positive
  • Possible parameter values could exist across a wide range
Example

If estimating the number of traffic accidents, we would probably choose a negative binomial. The parameters for the negative bionomial are \(\lambda\) and \(\kappa\), where \(\kappa\) is the inverse dispersion. For \(\kappa\), we need to set a prior, because we are not sure exactly which \(\kappa\) we should use. We know that \(\kappa\) must be positive, and does not affect the overall negative binomial distribution too much, so we would like a large variance. This is where the Log-Normal distribution shines!

PDF Equation:

$$ Pr(X=x|\mu, \sigma) = \frac{1}{x\sqrt{2\pi\sigma^2}} \exp \left( -\frac{(ln(x)-\mu)^2}{2\sigma^2} \right)$$

Covariance Distributions

LKJ Distribution
Lewandowski-Kurowicka-Joe Distribution
Description

A probability density function for a symmetric matrix. Commonly used as a prior for estimating covariance/correlation matrices.

Note: This function can usually not be visualised, as technically, the X-axis would contain a covariance/correlation matrix. In the simplified case above, a 2x2 correlation matrix has been used, as a 2x2 correlation only has 1 degree of freedom: the off-diagonal value. The diagonals must be 1. $$C = \begin{bmatrix} 1 & c_{ij} \\ c_{ij} & 1 \end{bmatrix}$$ This allows us to plot \(c_{ij}\) as a 1-dimensional X-axis.

Features:
  • Covariance/Correlation prior
  • (Practically) Bounded between -1, and 1
  • Continuous
  • Most useful in hierarchical models
  • Often preferred over the Wishart and Inverse-Wishart distributions
Example

Supposed we want to model comorbidity of certain diseases - that is, diseases occuring in the same person at the same time. These diseases may have some correlation, or maybe not. To do this, we need to specify a covariance, or correlation matrix. We can then generate the probability of this matrix occuring using the LKJ distribution.

PMF Equation:

$$ Pr(\mathsf{C}|\eta) =\left(2^{\sum_{k=1}^{K-1}(2(\eta-1)+K-k)(K-k)}\prod_{k=1}^{K-1}\left(B(\eta+(K-k-1)/2,\eta + (K-k-1)/2)\right)^{K-k}\right) (\mathrm{det}\,\mathsf{C})^{\eta-1}. $$ or $$ Pr(\mathsf{C}|\eta) \propto |\mathsf{C}^{\eta-1}|$$

Wishart Distribution
Wishart Distribution
Description

The Wishart distribution is a generalization to multiple dimensions of the gamma distribution. his distribution, along with the Inverse-Wishart and others, are of great importance in the estimation of covariance matrices in multivariate statistics.

Note: This function can usually not be visualised, as technically, the X-axis would contain a covariance/correlation matrix. In the simplified case above, a 2x2 correlation matrix has been used, as a 2x2 correlation only has 1 degree of freedom: the off-diagonal value. The diagonals must be 1. $$C = \begin{bmatrix} 1 & c_{ij} \\ c_{ij} & 1 \end{bmatrix}$$ This allows us to plot \(c_{ij}\) as a 1-dimensional X-axis.

Features:
  • Continuous
  • Prior distribution for covariance matrices
  • Conjugate to multivariate normal likelihoods
  • Present quite restrictive correlations between parameters. Often LKJ is preferred.
  • For a symmetric matrix
  • Practically bounded between -1 and 1
  • The Gamma distribution is a special case of this distribution
Related Distributions
  • Inverse-Wishart
  • Gamma Distribution
Example

Supposed we want to model comorbidity of certain diseases - that is, diseases occuring in the same person at the same time. These diseases may have some correlation, or maybe not. To do this, we need to specify a covariance, or correlation matrix. We can then generate the probability of this matrix occuring using the Wishart distribution.

PDF Equation:

$$ Pr(X=x|\Psi, n) = \frac{|x|^{(n-p-1)/2} \exp\left(-tr(\Psi^{-1}x) / 2\right)}{2^{\frac{np}{2}}|\Psi|^{\frac{n}{2}}\Gamma_p(\frac{n}{2})}$$ where \(x\) is a admissible covariance or correlation matrix, \(\Psi\) is a scale matrix, \(n\) are the degrees of freedom, and \(p\) are the number of rows of the \(x\) matrix.

The inverse Wishart distribution, also called the inverted Wishart distribution, is a probability distribution defined on real-valued positive-definite matrices.

Note: This function can usually not be visualised, as technically, the X-axis would contain a covariance/correlation matrix. In the simplified case above, a 2x2 correlation matrix has been used, as a 2x2 correlation only has 1 degree of freedom: the off-diagonal value. The diagonals must be 1. $$C = \begin{bmatrix} 1 & c_{ij} \\ c_{ij} & 1 \end{bmatrix}$$ This allows us to plot \(c_{ij}\) as a 1-dimensional X-axis.

Features:
  • Continuous
  • Prior distribution for covariance matrices
  • Conjugate to multivariate normal likelihoods
  • Present quite restrictive correlations between parameters. Often LKJ is preferred.
  • For a symmetric matrix
  • Practically bounded between -1 and 1
  • The Gamma distribution is a special case of this distribution

Supposed we want to model comorbidity of certain diseases - that is, diseases occuring in the same person at the same time. These diseases may have some correlation, or maybe not.
